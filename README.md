# Python README generator Project

This is a project where I have created a README.md file generator in Python, as a submission to the challenge Week 6 of the skills Bootcamp in Full Stack Web Development, organized by Step8Up Ltd.
The script can be run locally, with input prompt in the Terminal directly, ending with the creation of the corresponding README.md file in the same folder.
